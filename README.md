## 기능 설명

#### 게임

난수를 통한 벌레 당근의 움직임, 위치 랜덤 구현.<br/>
게임종료 후 다시시작시 초기화 구현.<br/>

#### 랭킹

웹이 로딩되면 fetch API GET을 통해 클라이언트가 현재 기록된 랭킹을 DB에서 가져옴<br/>
클라이언트가 게임을 이기면 스코어를 fetch API POST를 통해서 랭크 DB에 기록 <br/>

#### 사용자 콘솔

클라이언트가 슬라이더로 게임 난이도를 변경할 수 있도록 구현<br/>
